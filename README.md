# [바로가기](https://fir-shop-e5e79.web.app)

<div align=center>
  <h3>🌸소개</h3>
  <p>
   관리자계정으로 로그인하여 상품을 등록할 수 있습니다.<br />
   일반 사용자는 로그인하여 상품을 장바구니에 담아서 구매할 수 있습니다.<br />
   연결되어있는 결제 모듈은 테스트 모듈이며 결제시 돈이 빠져나가지 않습니다.
  </p>
</div>
